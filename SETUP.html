🚀 Quick Start

1
Navigate to Project Directory

Open your terminal/command prompt and go to:

cd C:\Users\aquarius12\fullplate-landing
2
Start a Local Server

Choose one of these commands:

python -m http.server 8000

Or if using Python 2:

python -m SimpleHTTPServer 8000

Or with Node.js:

npx http-server
3
View Your Site

Open your browser and visit:

http://localhost:8000
📝 Note: A local server is required for the parallax effects and all features to work properly. Simply opening the HTML file in a browser may cause issues with image loading and JavaScript functionality.

🔧 Customization Tips

Change the company quote? Edit line in index.html with class hero-quote

Update product descriptions? Find sections with class feature-description

Modify colors? Edit CSS variables in css/styles.css under :root {}

Add more sections? Use the existing HTML structure as a template
